# アプリ概要

**主要機能**
- 🀄 対局結果の記録・管理
- 📊 点数計算（ウマ・オカ対応、同点処理（上家取り・折半））
- 📱 4人以上の成績のサマリ機能

**技術スタック**
- **フレームワーク**: Flutter 3.8+
- **状態管理**: Riverpod
- **データベース**: SQLite
- **ルーティング**: GoRouter
